Save Urls

Program k uložení otevřených stránek na Chromu.

Tento kraťoušký pythonovský skript se hodí pro uložení aktuálně otevřených oken v prohlížeči MS Edge. Automatizuje něco, u čeho by se uživatel mohl uklikat u iniciace ukládání do oblíbených položek, volení správné složky atd. Mějmě např. otevřený následujících 13 stránek:

                  
                    import pyautogui
                    import os
                    import pyperclip
                    import time
                    
                    time.sleep(3)
                    
                    urls = []
                    os.system("start msedge www.google.com")
                    with open('urls.bat', 'a') as f:
                        while True:
                            pyautogui.hotkey('ctrl', 'tab')
                            pyautogui.hotkey('ctrl', 'l')
                            pyautogui.hotkey('ctrl', 'a')
                            pyautogui.hotkey('ctrl', 'c')
                            
                            url = pyperclip.paste()
                            if url == "www.google.com":
                                pass
                            elif url not in urls:
                                f.write('start msedge ' + url + '\n')
                                urls.append(url)
                                time.sleep(1)
                            else:
                                break
                    
                    print("Konec")
                    os.system("pause")

Po spuštění skriptu přes příkazovou řádku, program chvíli čeká, aby uživatel případně doaktivoval okno prohlížeče. Jinak totiž nebudou fungovat následující klávesové zkratky:

  • Ctrl + tab = mění aktivní okno
  • Ctrl + L = označí řádku s url adresou webu
  • Ctrl + A = označí celou url adresu
  • Ctrl + C = zkopíruje ji do schránky
Zkopírovanou adresu zhodnotí a pokud se ji ještě neuložil, uloží ji. Jinak se ukončí. Tak je zajištěno, že neběží while cyklus do nekonečna, protože takový je charakter klávesové zkratky Ctrl + tab.
